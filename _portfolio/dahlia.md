As part of my research under [Capra](https://capra.cs.cornell.edu/), I've made contributions to Dahlia, a programming language for designing hardware accelerators. It provides abstractions that guarantee predictable hardware realizability after type checking. The documentation can be found [here](https://capra.cs.cornell.edu/fuse/docs/overview/), and the Github repository can be found [here](https://github.com/cucapra/dahlia).
